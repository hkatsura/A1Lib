A1Lib is a simple editor for Korg A1 All Data System Exclusive files. Korg A1 Performance Signal Processor is a 2U rack digital signal processor introduced in 1991.

Requirements:
  • macOS 10.9.5 or later.
  • Korg A1 All Data System Exclusive files (.syx).
Features:
  • Supported file: Korg A1 All Data System Exclusive (.syx)
  • Open/Save .syx files
  • Copy/Paste or Drag/Drop patches from/to anywhere within a .syx file and from/to other .syx files
  • Edit the patch names
  • Copy/Paste or Drag/Drop map entries
  • Edit the map entries
  • Change the MIDI channel
  • Change the utility parameters such as the emphasis mode and the pedal assignments
  • Print a list of patch names in the .syx file

Installation:
Drag and drop the application icon to the Applications folder on your hard drive.
Instructions:
First, you need a .syx file. You can either download it from korgaseries.info or from your Korg A1 via a MIDI interface and a librarian application such as SysEx Librarian (available at www.snoize.com).

To open a Korg A1 All Data System Exclusive .syx file, choose the "Open..." menu item from the File menu, choose a .syx file, and click the Open button.

After rearranging patches and editing patch names, choose the "Save" or "Save As..." menu item from the File menu to save the changes.

To upload the changes to your Korg A1, use a MIDI interface and a librarian application such as SysEx Librarian.

To print a list of patch names, choose the "Print..." menu item from the File menu, and click the Print button.
